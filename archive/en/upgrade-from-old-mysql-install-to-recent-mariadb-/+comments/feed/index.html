MariaDB Knowledge Base Comments for: Upgrade from old Mysql install to recent Mariadb.https://mariadb.com/kb/en/upgrade-from-old-mysql-install-to-recent-mariadb-/+comments/feed/en-usTue, 09 Jan 2024 00:21:01 +0000Re: Upgrade from old Mysql install to recent Mariadb.https://mariadb.com/kb/en/upgrade-from-old-mysql-install-to-recent-mariadb-/+comments/6553<p>When you say starts to fail at the user database, this is probably because the MySQL 5.7 users are using SHA-256 - you can check which MySQL users are using SHA-256 by executing:</p> <pre class="fixed">SELECT user, plugin FROM mysql.user where plugin like &#34;%sha%&#34;; </pre><p>You can change the user/s to use a protocol compatible with both MySQL and MariaDB with:</p> <pre class="fixed">ALTER USER user_name IDENTIFIED WITH mysql_native_password BY 'new_password'; </pre><p>This can all be done on the database side, and there should be no need to change anything on the application side.</p> Ian Gilfillanhttps://mariadb.com/kb/en/upgrade-from-old-mysql-install-to-recent-mariadb-/+comments/6553
